Bases: object

Code author: Wilfried Mercier - IRAP/LAM <wilfried.mercier@lam.fr>

A general named column to assicate to an Enum object.

Arguments

Parameters:

  • name (str) – name of the column when included in a Table

  • unit (str) – unit of the column

Keyword arguments

Parameters:

  • end (str) – end string to append to the associate enum name

  • log (bool) – whether this column holds log values. This will trigger a conversion to the power of 10 if True.

Raises:

TypeError – if name, unit and end are not of type str
